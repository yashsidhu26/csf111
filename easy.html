The struct Student is passed by value into the function. You can access its fields using the dot operator, e.g. s.name, s.roll, s.marks[0].
2

void displayAverage(Student s) {

    int total = s.marks[0] + s.marks[1] + s.marks[2];

    // Step 2: Divide by 3.0 (not 3) so we get a decimal result
    float avg = total / 3.0;

    // Step 3: Print everything using printf
    // %s = string, %d = integer, %.2f = float with 2 decimal places
    printf("Student: %s, Roll: %d, Average: %.2f\n",
           s.name, s.roll, avg);

}
Example Output
Original array: 10 20 30 40 50 Reversed array: 50 40 30 20 10
💡 How to think about this
1
Start with two variables: left = 0 (points to the first element) and right = n - 1 (points to the last element).
2
Swap arr[left] and arr[right] using a temporary variable temp. This is the classic 3-step swap: save → overwrite → restore.
3
Move left one step right (left++) and right one step left (right--), then repeat.
4
Stop when left >= right. At that point the two sides have crossed the middle and the entire array is reversed.
📌 For array {10, 20, 30, 40, 50}: swap(10,50) → swap(20,40) → middle 30 stays. Done!
c — Solution
void reverseArray(int* arr, int n) {
    int left  = 0;       // starts at the beginning
    int right = n - 1;   // starts at the end

    // Keep swapping until the two sides meet
    while (left < right) {
        // Classic 3-step swap using a temp variable
        int temp    = arr[left];
        arr[left]   = arr[right];
        arr[right]  = temp;

        left++;   // move left pointer forward
        right--;  // move right pointer backward
    }
    // When left >= right the whole array is reversed
}

💡 How to think about this
1
The function receives int* max and int* min — these are pointers to the max and min variables in main(). Writing to *max (dereference) actually changes the variable in the caller.
2
Start by assuming the first element is both the biggest and smallest: *max = arr[0] and *min = arr[0].
3
Loop through the rest of the array (starting at index 1). For each element, if it's bigger than the current max, update *max. If smaller than current min, update *min.
📌 The * in front of a pointer variable means "the value at this address". Writing *max = 10 stores 10 at the address that max points to — i.e., the caller's variable.
c — Solution
void findMaxMin(int* arr, int n, int* max, int* min) {
    // Assume the first element is both max and min to start with
    *max = arr[0];
    *min = arr[0];

    // Check every other element
    for (int i = 1; i < n; i++) {
        if (arr[i] > *max) {
            *max = arr[i];  // found a new maximum
        }
        if (arr[i] < *min) {
            *min = arr[i];  // found a new minimum
        }
    }
    // After the loop, *max and *min hold the answers.
    // Because max and min are pointers, main() can see the values we wrote.
}

Example Output

💡 How to think about this
1
realloc(ptr, newBytes) resizes memory that was previously allocated by malloc. It automatically copies the old data to the new (possibly different) memory location and returns the new pointer.
2
Always save the result of realloc into a temporary variable first. If realloc fails it returns NULL — if you overwrite your original pointer with NULL you'll lose the only reference to your old array and have a memory leak.
3
After the resize, the new slots (from index oldSize to newSize-1) are uninitialized — the caller will fill them in, which is already handled in main().
💡 Think of realloc as "expand the room". The furniture (old data) stays, and new empty space is added at the end.
c — Solution
int* resizeArray(int* arr, int oldSize, int newSize) {
    // realloc resizes the block that arr points to.
    // It copies old data automatically and returns a new pointer.
    // We save it in a new variable first in case realloc fails.
    int* newArr = (int*) realloc(arr, newSize * sizeof(int));

    // If realloc failed, it returns NULL
    if (newArr == NULL) {
        printf("Memory reallocation failed!\n");
        free(arr);   // free the old memory before exiting
        exit(1);
    }

    // Return the new (possibly relocated) pointer
    return newArr;
}

💡 How to think about this
1
insertEnd: First allocate a new node with malloc. Set its data to the value and its next to NULL (it has no successor yet).
2
Special case: if head == NULL the list is empty, so the new node becomes the head. Return it immediately.
3
Otherwise, walk a curr pointer all the way to the last node (the one whose next is NULL), then set curr->next = newNode to attach it. Return the unchanged head.
4
printList: Walk from head to the end. Print each node's data followed by " -> ". After the loop, print "NULL" to mark the end of the chain.
📌 A singly linked list is a chain of boxes. Each box holds data and an arrow pointing to the next box. The last box's arrow points to NULL meaning "end of list".
c — Solution
Node* insertEnd(Node* head, int value) {
    // Step 1: Create a new node on the heap
    Node* newNode = (Node*) malloc(sizeof(Node));
    newNode->data = value;
    newNode->next = NULL;   // this will be the new last node

    // Step 2: If the list is empty, the new node is the head
    if (head == NULL) {
        return newNode;
    }

    // Step 3: Walk to the very last node in the list
    Node* curr = head;
    while (curr->next != NULL) {
        curr = curr->next;   // move one step forward
    }

    // Step 4: Attach the new node at the end
    curr->next = newNode;

    return head;  // head itself did not change
}

void printList(Node* head) {
    Node* curr = head;
    // Visit every node until we reach NULL
    while (curr != NULL) {
        printf("%d -> ", curr->data);
        curr = curr->next;
    }
    printf("NULL\n");  // marks the end of the list
}

Example Output

💡 How to think about this
1
Start a count variable at 0 and a curr pointer at head.
2
As long as curr is not NULL, there's a node to count. Increment count by 1, then advance curr to curr->next.
3
When curr becomes NULL we've passed the last node. Return count. If the list was empty, curr started as NULL, so the loop never ran and we return 0 automatically.
📌 This is the most common linked list traversal pattern. You'll use it in almost every linked list function: start at head → advance until NULL → do something each step.
c — Solution
int countNodes(Node* head) {
    int count = 0;       // will hold our running total
    Node* curr = head;   // start from the beginning

    // Visit each node until we reach the end (NULL)
    while (curr != NULL) {
        count++;             // found one more node
        curr = curr->next;   // move to the next node
    }

    return count;
    // If list was empty (head == NULL), count is still 0 — correct!
}

💡 How to think about this
1
Empty list check: If head == NULL there's nothing to delete. Return NULL immediately.
2
Deleting the head: If the first node holds the key (head->data == key), save the next node, free the head, and return the next node as the new head. This is a special case because we can't "re-wire" from before the head.
3
Deleting a middle or last node: Walk curr until curr->next holds the key. Then set curr->next = curr->next->next to skip over the node to delete. Free the skipped node.
4
If the key doesn't exist in the list, the loop ends without finding it — simply return the head unchanged.
📌 Deleting a node is like removing a link from a chain. You re-wire the link before it to point to the link after it, then remove the middle link.
c — Solution
Node* deleteValue(Node* head, int key) {
    // Case 1: List is empty — nothing to do
    if (head == NULL) {
        return NULL;
    }

    // Case 2: The node to delete is the head node
    if (head->data == key) {
        Node* newHead = head->next;   // save the next node
        free(head);                   // delete the old head
        return newHead;               // return new head to main()
    }

    // Case 3: Find the node BEFORE the one we want to delete
    Node* curr = head;
    while (curr->next != NULL) {
        if (curr->next->data == key) {
            // Found it! curr->next is the node to delete.
            Node* toDelete = curr->next;
            curr->next = toDelete->next;  // skip over it
            free(toDelete);               // release its memory
            return head;                  // head didn't change
        }
        curr = curr->next;  // move forward
    }

    // Case 4: Key was never found — return unchanged head
    return head;
}

void printList(Node* head) {
    Node* curr = head;
    while (curr != NULL) {
        printf("%d -> ", curr->data);
        curr = curr->next;
    }
    printf("NULL\n");
}

2
When the loop ends (when curr reaches NULL), sum holds the total of all values. Return it.
3
Empty list case is handled automatically: the loop never runs, so sum stays 0 and we return 0.
📌 This is essentially the same pattern as countNodes, just replacing count++ with sum += curr->data. Recognising this shared pattern helps you build all linked list functions quickly.
c — Solution
int sumList(Node* head) {
    int sum = 0;         // total starts at zero
    Node* curr = head;   // start at the first node

    // Add up every node's data value
    while (curr != NULL) {
        sum += curr->data;   // same as: sum = sum + curr->data
        curr = curr->next;   // move to the next node
    }

    return sum;
    // If list was empty (head == NULL), we return 0 — correct!
}

💡 How to think about this
1
Row sums: The outer loop picks a row (i). The inner loop goes across all columns (j) of that row, adding mat[i][j] to a sum variable. After the inner loop, print the sum for that row, then reset sum = 0 for the next row.
2
Column sums: The logic is exactly the same but with the loops flipped. The outer loop picks a column (j). The inner loop goes down all rows (i) of that column, collecting mat[i][j].
3
The key insight: in a 2D array mat[i][j], i is the row and j is the column. To sum a row, keep i fixed and vary j. To sum a column, keep j fixed and vary i.
📌 Reset sum = 0 before each new row/column sum, otherwise you'll be adding values from the previous row/column!
c — Solution
void rowSum(int mat[3][3]) {
    printf("Row sums:\n");
    // Outer loop = which row we're summing
    for (int i = 0; i < 3; i++) {
        int sum = 0;   // reset sum for each new row
        // Inner loop = go through all columns of this row
        for (int j = 0; j < 3; j++) {
            sum = sum + mat[i][j];
        }
        printf("Row %d: %d\n", i, sum);
    }
}

void colSum(int mat[3][3]) {
    printf("\nColumn sums:\n");
    // Outer loop = which column we're summing
    for (int j = 0; j < 3; j++) {
        int sum = 0;   // reset sum for each new column
        // Inner loop = go down all rows of this column
        for (int i = 0; i < 3; i++) {
            sum = sum + mat[i][j];
        }
        printf("Col %d: %d\n", j, sum);
    }
}

    // TODO
}
Example Output
Original matrix: 1 2 3 4 5 6 7 8 9 Transposed matrix: 1 4 7 2 5 8 3 6 9
💡 How to think about this
1
Transposing means flipping along the main diagonal. Element at [i][j] goes to [j][i]. For each pair (i, j), swap mat[i][j] and mat[j][i].
2
We only do this for positions where i < j (upper triangle). If we swapped for all pairs including i > j, we'd just undo the swap we already did — ending up with the original matrix!
3
The diagonal elements (i == j) never move, so we skip them. The inner loop j = i + 1 naturally skips both the diagonal and the lower triangle.
4
After swapping, print the matrix using the same nested loop pattern.
📌 Why j = i + 1 and not j = 0? Because if you swap (0,1) and later swap (1,0), you just swap back. Starting j at i+1 ensures each pair is swapped exactly once.
c — Solution
void transpose(int mat[3][3]) {
    // Step 1: Swap elements across the diagonal
    // We only go through the "upper triangle" (j > i)
    // to avoid swapping the same pair twice
    for (int i = 0; i < 3; i++) {
        for (int j = i + 1; j < 3; j++) {
            // Swap mat[i][j] and mat[j][i]
            int temp    = mat[i][j];
            mat[i][j]   = mat[j][i];
            mat[j][i]   = temp;
        }
    }

    // Step 2: Print the now-transposed matrix
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", mat[i][j]);
        }
        printf("\n");
    }
}

💡 How to think about this
1
createMatrix: An int** is a pointer to pointers. First, allocate an array of rows pointers: malloc(rows * sizeof(int*)). Then, for each row, allocate an array of cols integers: malloc(cols * sizeof(int)). Think of it as: allocate the row headers, then allocate each row's actual data.
2
fillMatrix: Two nested loops — outer for rows, inner for cols. Use scanf("%d", &mat[i][j]). Even though mat is int**, you can still use mat[i][j] and C handles the double-dereferencing for you.
3
printMatrix: Same nested loop, just printing with printf. Print a newline \n after the end of each row.
4
freeMatrix: Free in reverse order of allocation. First free each individual row array (free(mat[i])). After all rows are freed, free the outer pointer array (free(mat)). Doing it in the wrong order would be a bug.
📌 A dynamic 2D array is really just an array of 1D arrays. Visualise it as: mat → [ptr0, ptr1, ptr2] where ptr0 → [row0 data], ptr1 → [row1 data], etc.
c — Solution
int** createMatrix(int rows, int cols) {
    // Step 1: Allocate an array of 'rows' pointers
    // Each pointer will later point to one row of ints
    int** mat = (int**) malloc(rows * sizeof(int*));

    // Step 2: Allocate each row individually
    for (int i = 0; i < rows; i++) {
        mat[i] = (int*) malloc(cols * sizeof(int));
        // mat[i] now points to an array of 'cols' integers
    }

    return mat;  // return the 2D structure to the caller
}

void fillMatrix(int** mat, int rows, int cols) {
    printf("Enter elements:\n");
    // Read one integer at a time using row then column
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            scanf("%d", &mat[i][j]);  // works just like a normal 2D array
        }
    }
}

void printMatrix(int** mat, int rows, int cols) {
    printf("Matrix:\n");
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            printf("%d ", mat[i][j]);
        }
        printf("\n");  // new line after each row
    }
}

void freeMatrix(int** mat, int rows) {
    // Step 1: Free each row's data array first
    for (int i = 0; i < rows; i++) {
        free(mat[i]);   // free the ints in row i
    }
    // Step 2: Then free the outer pointer array
    free(mat);
    // Rule: always free in the OPPOSITE order you allocated
}
