Student Course Management System
StructuresDynamic MemoryTough
Story
You're building a student portal where each student can enroll in multiple courses added dynamically. The struct stores a char* name and a char** course list that grows one course at a time using realloc.

  • Implement Student createStudent(const char* name) — heap-allocate the name with strdup, set courses = NULL, courseCount = 0

💡 How to think about this
1
createStudent: The struct has a char* name (heap string) and a char** courses (heap array of heap strings). On creation, courses is NULL and courseCount is 0 — no courses yet.
2
addCourse: Each time we add a course, we need one more slot in the courses array. Use realloc(s->courses, (s->courseCount + 1) * sizeof(char*)) to grow the array by one, then strdup the course name into the new slot.
3
freeStudent: Free in reverse order of allocation — first free each course string, then free the courses array, then free the name. Setting freed pointers to NULL afterwards is good practice.
📌 The key challenge here is that realloc on a NULL pointer behaves exactly like malloc — so the very first addCourse call works correctly without any special case.
c — Solution
Student createStudent(const char* name) {
    Student s;
    s.name = strdup(name);   // heap-copy the name
    s.courses = NULL;        // no courses yet
    s.courseCount = 0;
    return s;
}

void addCourse(Student* s, const char* course) {
    // Grow the courses array by one slot
    // realloc(NULL, size) is the same as malloc(size) — handles the first call
    s->courses = (char**) realloc(s->courses,
                     (s->courseCount + 1) * sizeof(char*));

    // Store a heap copy of the course name in the new slot
    s->courses[s->courseCount] = strdup(course);
    s->courseCount++;
}

void printStudent(Student* s) {
    printf("Student: %s\n", s->name);
    printf("Courses:\n");
    for (int i = 0; i < s->courseCount; i++) {
        printf("  - %s\n", s->courses[i]);
    }
}

void freeStudent(Student* s) {
    // Step 1: free each individual course string
    for (int i = 0; i < s->courseCount; i++) {
        free(s->courses[i]);
    }
    // Step 2: free the array of course pointers
    free(s->courses);
    // Step 3: free the name string
    free(s->name);

    // Reset to safe state
    s->courses = NULL;
    s->name = NULL;
    s->courseCount = 0;
}

💡 How to think about this
1
The matrix is allocated with calloc — this zero-initialises every NameRow, meaning names = NULL and count = 0 to start. This is why realloc(NULL, ...) works as the first call in addName.
2
addName: Same pattern as Q1's addCourse — realloc to grow the names array by one, strdup to copy the name string into the new slot.

c — Solution
void addName(NameRow* row, const char* name) {
    // Grow the names array by one
    row->names = (char**) realloc(row->names,
                     (row->count + 1) * sizeof(char*));
    // Copy the name string onto the heap
    row->names[row->count] = strdup(name);
    row->count++;
}

void printRows(NameRow* matrix, int rows) {
    for (int i = 0; i < rows; i++) {
        printf("Row %d: ", i);
        for (int j = 0; j < matrix[i].count; j++) {
            printf("%s ", matrix[i].names[j]);
        }
        printf("\n");
    }
}

void freeRows(NameRow* matrix, int rows) {
    for (int i = 0; i < rows; i++) {
        // Free each name string in this row
        for (int j = 0; j < matrix[i].count; j++) {
            free(matrix[i].names[j]);
        }
        // Free the array of name pointers
        free(matrix[i].names);
    }
    // Note: the outer 'matrix' array itself is freed by the caller
}

💡 How to think about this
1
A void* can hold a pointer to anything. To store an int, we allocate a heap int and point to it: int* p = malloc(sizeof(int)); *p = value;. To store a string, we use strdup. Both produce a void* we can store in data.
2
createNode: Allocate the node. If type is INT_TYPE, allocate an int on the heap, dereference the incoming void* as int* to copy the value. If STRING_TYPE, call strdup((char*)data). Store the result in node->data.
3
printList: Check curr->type. If INT, cast curr->data to int* and dereference: *(int*)curr->data. If STRING, cast to char* and print directly.
4
freeList: For each node, free node->data (works for both — both were heap-allocated), then free the node itself.
📌 The cast pattern *(int*)ptr means: "treat ptr as a pointer-to-int, then dereference it to get the int value". This is how you safely extract typed values from a void*.
c — Solution
Node* createNode(void* data, DataType type) {
    Node* node = (Node*) malloc(sizeof(Node));
    node->type = type;
    node->next = NULL;

    if (type == INT_TYPE) {
        // Allocate a new int on the heap and copy the value
        int* intData = (int*) malloc(sizeof(int));
        *intData = *(int*)data;   // dereference incoming void* as int*
        node->data = intData;
    } else {
        // strdup makes a heap copy of the string
        node->data = strdup((char*)data);
    }
    return node;
}

void printList(Node* head) {
    Node* curr = head;
    while (curr != NULL) {
        if (curr->type == INT_TYPE) {
            printf("INT: %d\n", *(int*)curr->data);
        } else {
            printf("STRING: %s\n", (char*)curr->data);
        }
        curr = curr->next;
    }
}

void freeList(Node* head) {
    Node* curr = head;
    while (curr != NULL) {
        Node* next = curr->next;   // save before freeing
        free(curr->data);          // free the int or string
        free(curr);                // free the node itself
        curr = next;
    }
}

💡 How to think about this
1
Allocate rows int-pointers first. Then for each row, allocate cols ints. This is the standard two-step dynamic 2D allocation.
2
populateMatrix: Use a single counter variable starting at 1. Nested loops: outer for rows, inner for cols. Assign mat[i][j] = counter++ each iteration.
3
printMatrix: Use %3d format specifier for right-aligned numbers in a fixed-width column. Print a newline after each row.
4
freeMatrix: Free each row array first, then the outer pointer array. Never reverse this order.
📌 %3d in printf right-aligns an integer in a field of width 3, making the matrix columns line up neatly regardless of how many digits each number has.
c — Solution
int** createMatrix(int rows, int cols) {
    int** mat = (int**) malloc(rows * sizeof(int*));
    for (int i = 0; i < rows; i++) {
        mat[i] = (int*) malloc(cols * sizeof(int));
    }
    return mat;
}

void populateMatrix(int** mat, int rows, int cols) {
    int counter = 1;
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            mat[i][j] = counter++;   // fills 1, 2, 3, 4, 5, ...
        }
    }
}

void printMatrix(int** mat, int rows, int cols) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            printf("%3d ", mat[i][j]);   // %3d = right-align in width-3 field
        }
        printf("\n");
    }
}

void freeMatrix(int** mat, int rows) {
    for (int i = 0; i < rows; i++) {
        free(mat[i]);   // free each row's int array
    }
    free(mat);          // free the outer pointer array
}

Example Output
Reversed: elpmaxe
💡 How to think about this
1
Use strlen(str) to find the length. Set left = 0 and right = len - 1.
2
While left < right, swap str[left] and str[right] using a char temp variable. Then increment left and decrement right.
3
This is identical to the integer array reversal from Easy Q3, just with char instead of int. For an empty string or single character, the condition is false from the start and nothing happens — correct.
📌 This is the same two-pointer swap pattern used in array reversal. The only difference is the element type is char instead of int. The logic is identical.
c — Solution
void reverse(char* str) {
    int left  = 0;
    int right = strlen(str) - 1;   // index of last char (not '\0')

    while (left < right) {
        // Swap characters at left and right
        char temp   = str[left];
        str[left]   = str[right];
        str[right]  = temp;

        left++;    // move inward from the left
        right--;   // move inward from the right
    }
    // When left >= right we've swapped everything — string is reversed
}

💡 How to think about this
1
Both addStudent and addCourse use the same "insert at front" pattern: allocate a new node, strdup the name/course, set next to the current head, return the new node as the new head. This is simpler than inserting at the tail.

c — Solution
StudentNode* addStudent(StudentNode* head, const char* name) {
    StudentNode* s = (StudentNode*) malloc(sizeof(StudentNode));
    s->name    = strdup(name);
    s->courses = NULL;   // no courses yet
    s->next    = head;   // insert at front
    return s;            // new head
}

CourseNode* addCourse(CourseNode* head, const char* course) {
    CourseNode* c = (CourseNode*) malloc(sizeof(CourseNode));
    c->course = strdup(course);
    c->next   = head;   // insert at front
    return c;
}

    StudentNode* s = head;
    while (s != NULL) {
        printf("Student: %s\n  Courses: ", s->name);
        CourseNode* c = s->courses;
        while (c != NULL) {
            printf("%s", c->course);
            if (c->next) printf(", ");
            c = c->next;
        }
        printf("\n");
        s = s->next;
    }
}

void freeAll(StudentNode* head) {
    StudentNode* s = head;
    while (s != NULL) {
        // Step 1: free this student's course linked list
        CourseNode* c = s->courses;
        while (c != NULL) {
            CourseNode* nextC = c->next;   // save before free
            free(c->course);               // free the course string
            free(c);                       // free the course node
            c = nextC;
        }

        // Step 2: free the student itself
        StudentNode* nextS = s->next;   // save before free
        free(s->name);                  // free the name string
        free(s);                        // free the student node
        s = nextS;
    }
}

c — Solution
// Sort a single 1D array using selection sort
void selectionSort(int* arr, int n) {
    for (int i = 0; i < n - 1; i++) {
        // Find the index of the minimum in arr[i..n-1]
        int minIdx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIdx]) {
                minIdx = j;
            }
        }
        // Swap arr[i] with the found minimum
        int temp      = arr[i];
        arr[i]        = arr[minIdx];
        arr[minIdx]   = temp;
    }
}

int main() {
    int rows = 3;
    int* colCounts = (int*) malloc(rows * sizeof(int));
    int** jagged   = (int**) malloc(rows * sizeof(int*));

    // Set column counts and allocate each row
    colCounts[0] = 3;    colCounts[1] = 2;    colCounts[2] = 4;

    // Fill with unsorted values
    int initData[3][4] = {{5, 1, 3, 0}, {4, 2, 0, 0}, {9, 6, 8, 7}};
    for (int i = 0; i < rows; i++) {
        jagged[i] = (int*) malloc(colCounts[i] * sizeof(int));
        for (int j = 0; j < colCounts[i]; j++) {
            jagged[i][j] = initData[i][j];
        }
    }

    // Sort each row independently
    for (int i = 0; i < rows; i++) {
        selectionSort(jagged[i], colCounts[i]);
    }

    // Print the sorted jagged array
    for (int i = 0; i < rows; i++) {
        printf("Row %d: ", i);
        for (int j = 0; j < colCounts[i]; j++) {
            printf("%d ", jagged[i][j]);
        }
        printf("\n");
        free(jagged[i]);   // free each row
    }

    free(colCounts);
    free(jagged);
    return 0;
}

💡 How to think about this
1
intCompare: Cast both void* args to int* and dereference to compare. Return negative if a<b, zero if equal, positive if a>b.
2
strCompare: Cast both to char** (array of string pointers), dereference to get char*, then call strcmp.
3
swap: Use a temporary char buffer of size bytes. Copy size bytes from a to temp, from b to a, from temp to b using memcpy.
4
sortAndPrint: Use the type to choose element size (sizeof(int) or sizeof(char*)) and the compare function. Run selection sort using pointer arithmetic to access each element.
📌 Generic sorting works by treating all data as raw bytes (char*) and using element size + a compare function to navigate the array. (char*)arr + i * elemSize gives you a pointer to element i.
c — Solution
int intCompare(const void* a, const void* b) {
    return *(int*)a - *(int*)b;
}

int strCompare(const void* a, const void* b) {
    return strcmp(*(char**)a, *(char**)b);
}

void swap(void* a, void* b, size_t size) {
    char temp[256];        // temporary buffer (big enough)
    memcpy(temp, a, size); // temp = *a
    memcpy(a, b, size);    // *a = *b
    memcpy(b, temp, size); // *b = temp
}

void sortAndPrint(GenericArray* gArr) {
    size_t elemSize = (gArr->type == INT_ARRAY)
                      ? sizeof(int)
                      : sizeof(char*);

    int (*cmp)(const void*, const void*) = (gArr->type == INT_ARRAY)
                                           ? intCompare
                                           : strCompare;

    char* base = (char*) gArr->arr;   // treat as byte array

    // Selection sort
    for (int i = 0; i < gArr->size - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < gArr->size; j++) {
            if (cmp(base + j * elemSize, base + minIdx * elemSize) < 0) {
                minIdx = j;
            }
        }
        if (minIdx != i) {
            swap(base + i * elemSize, base + minIdx * elemSize, elemSize);
        }
    }

    // Print
    if (gArr->type == INT_ARRAY) {
        printf("Sorted ints: ");
        for (int i = 0; i < gArr->size; i++)
            printf("%d ", ((int*)gArr->arr)[i]);
    } else {
        printf("Sorted strings: ");
        for (int i = 0; i < gArr->size; i++)
            printf("%s ", ((char**)gArr->arr)[i]);
    }
    printf("\n");
}

Example Output
1 2 3 4 5 6
💡 How to think about this
1
There are 3 levels: the outer int*** is an array of row pointers. Each row (int**) is an array of cell pointers. Each cell (int*) points to a single heap integer.
2
Allocation order: matrix = malloc(rows * sizeof(int**)) → for each row: matrix[i] = malloc(cols * sizeof(int*)) → for each cell: matrix[i][j] = malloc(sizeof(int)).
3
Assign values: *matrix[i][j] = value. Print: printf("%d", *matrix[i][j]). Free in reverse: first each individual int, then each row array, then the outer matrix.
📌 Think of it as 3 floors: floor 3 (outer) → floor 2 (rows) → floor 1 (individual ints). Build top-down, free bottom-up: free the ints first, then the row arrays, then the outer array.
c — Solution
int main() {
    int rows = 2, cols = 3;

    // Level 1: array of row pointers (int**)
    int*** matrix = (int***) malloc(rows * sizeof(int**));

    int counter = 1;
    for (int i = 0; i < rows; i++) {
        // Level 2: array of cell pointers (int*) for each row
        matrix[i] = (int**) malloc(cols * sizeof(int*));

        for (int j = 0; j < cols; j++) {
            // Level 3: a single heap-allocated int for each cell
            matrix[i][j] = (int*) malloc(sizeof(int));
            *matrix[i][j] = counter++;   // assign value
        }
    }

    // Print: dereference twice to get the int value
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            printf("%3d ", *matrix[i][j]);
        }
        printf("\n");
    }

    // Free in reverse order: int → row → outer
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            free(matrix[i][j]);   // free each individual int
        }
        free(matrix[i]);          // free each row array
    }
    free(matrix);                 // free the outer array
    return 0;
}

Insert one string into another at a given position, returning a brand-new dynamically allocated result string. The originals are not modified. E.g., inserting "Amazing" at position 5 in "HelloWorld" gives "HelloAmazingWorld".

// Implement insertString here
Example Output
Result: HelloAmazingWorld
💡 How to think about this
1
Allocate a result buffer of size strlen(base) + strlen(insert) + 1 (the +1 is for the null terminator).
2
Build the result in three steps using strncpy and strcat: (a) copy the first pos characters of base into result; (b) concatenate the insert string; (c) concatenate the remainder of base starting at base + pos.
3
After strncpy(result, base, pos), manually set result[pos] = '\0' because strncpy does not null-terminate if the source is longer than the count. Then strcat works correctly.
⚠️ strncpy does NOT guarantee null-termination if the copy count is less than the source length. Always follow it with result[pos] = '\0' before using strcat.
c — Solution
char* insertString(const char* base, const char* insert, int pos) {
    int baseLen   = strlen(base);
    int insertLen = strlen(insert);

    // Allocate exactly enough for the combined string + null terminator
    char* result = (char*) malloc(baseLen + insertLen + 1);

    // Step 1: copy the first 'pos' characters of base
    strncpy(result, base, pos);
    result[pos] = '\0';   // IMPORTANT: strncpy doesn't always null-terminate

    // Step 2: append the insert string
    strcat(result, insert);

    // Step 3: append the rest of base (starting after position 'pos')
    strcat(result, base + pos);

    return result;   // caller must free() this
}

Example Output
Department: Computer Science Student: Alice Courses: OS, Networks Student: Bob Courses: Algorithms
💡 How to think about this
1
This combines Q1 (student + dynamic course array) with Q6 (students in a linked list). addStudent allocates a new Student node, strdups the name, sets courses=NULL, courseCount=0, and inserts at the front of dept->students.
2
addCourse is identical to Q1's addCourse — realloc the courses array and strdup the new course into the new slot.
3
freeDepartment: Walk the student linked list. For each student: free all course strings → free the courses array → free the student name → save next → free the student node. The deptName is a string literal here so we don't free it, but if it were heap-allocated we would.
📌 This question is essentially Q1 + Q6 combined. Recognising these sub-patterns makes the problem much less overwhelming — solve each part separately.
c — Solution
void addStudent(Department* dept, const char* name) {
    Student* s = (Student*) malloc(sizeof(Student));
    s->name        = strdup(name);
    s->courses     = NULL;
    s->courseCount = 0;
    s->next        = dept->students;  // insert at front
    dept->students = s;
}

void addCourse(Student* student, const char* course) {
    // Grow the courses array by one slot
    student->courses = (char**) realloc(student->courses,
                           (student->courseCount + 1) * sizeof(char*));
    student->courses[student->courseCount] = strdup(course);
    student->courseCount++;
}

void printDepartment(Department* dept) {
    printf("Department: %s\n", dept->deptName);
    Student* s = dept->students;
    while (s != NULL) {
        printf("  Student: %s\n    Courses: ", s->name);
        for (int i = 0; i < s->courseCount; i++) {
            printf("%s", s->courses[i]);
            if (i < s->courseCount - 1) printf(", ");
        }
        printf("\n");
        s = s->next;
    }
}

void freeDepartment(Department* dept) {
    Student* s = dept->students;
    while (s != NULL) {
        // Free all course strings
        for (int i = 0; i < s->courseCount; i++) {
            free(s->courses[i]);
        }
        free(s->courses);  // free the courses array
        free(s->name);     // free the student's name

        Student* next = s->next;  // save before free
        free(s);                  // free the student node
        s = next;
    }
    dept->students = NULL;
}

💡 How to think about this
1
To store an int in a void*: allocate a heap int, write your value, store its address. Same for float. For strings: use strdup. All three produce a void* pointer to heap data.
2
printCell: Switch on c->type. For INT: printf("%d", *(int*)c->data). For FLOAT: printf("%.2f", *(float*)c->data). For STRING: printf("%s", (char*)c->data).

c — Solution
void printCell(Cell* c) {
    switch (c->type) {
        case INT:    printf("%d ",   *(int*)c->data);   break;
        case FLOAT:  printf("%.2f ", *(float*)c->data); break;
        case STRING: printf("%s ",   (char*)c->data);   break;
    }
}

void freeCell(Cell* c) {
    free(c->data);   // works for all types — all were heap-allocated
}

int main() {
    int rows = 2, cols = 3;

        matrix[i] = (Cell*) malloc(cols * sizeof(Cell));

    int* vi = malloc(sizeof(int));     *vi = 42;
    matrix[0][0].data = vi;  matrix[0][0].type = INT;

    float* vf = malloc(sizeof(float)); *vf = 3.14f;
    matrix[0][1].data = vf;  matrix[0][1].type = FLOAT;

    matrix[0][2].data = strdup("hello"); matrix[0][2].type = STRING;

    // Populate row 1
    int* vi2 = malloc(sizeof(int));    *vi2 = 7;
    matrix[1][0].data = vi2; matrix[1][0].type = INT;

        for (int j = 0; j < cols; j++) freeCell(&matrix[i][j]);

💡 How to think about this
1
jagged is an array of Node* pointers — each element is the head of a linked list. calloc zero-initialises all heads to NULL (empty lists to start).
2
addNode: Standard insert-at-end. Allocate a new node, set data, next=NULL. If head is NULL return new node. Otherwise walk to tail and attach.
3
printJagged: For each row index i, walk the linked list at rows[i] and print each node's data in arrow format.
4
freeJagged: For each row, walk the linked list and free every node (save next before freeing). The jagged array itself is freed by the caller.
📌 This data structure gives you the best of both worlds: O(1) random access to any row (via array index), and variable-length rows (via linked list). It's used in graph adjacency lists.
c — Solution
Node* addNode(Node* head, int value) {
    Node* newNode = (Node*) malloc(sizeof(Node));
    newNode->data = value;
    newNode->next = NULL;

    if (head == NULL) return newNode;  // empty list

    // Walk to the tail
    Node* curr = head;
    while (curr->next != NULL) curr = curr->next;
    curr->next = newNode;
    return head;
}

void printJagged(Node** rows, int nRows) {
    for (int i = 0; i < nRows; i++) {
        printf("Row %d: ", i);
        Node* curr = rows[i];
        while (curr != NULL) {
            printf("%d -> ", curr->data);
            curr = curr->next;
        }
        printf("NULL\n");
    }
}

void freeJagged(Node** rows, int nRows) {
    for (int i = 0; i < nRows; i++) {
        Node* curr = rows[i];
        while (curr != NULL) {
            Node* next = curr->next;   // SAVE next before freeing
            free(curr);
            curr = next;
        }
        rows[i] = NULL;   // mark as freed
    }
    // The outer jagged array is freed by the caller
}
